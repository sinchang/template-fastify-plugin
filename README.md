> Scaffolding out a fastify plugin template

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao fastify-plugin my-project
```

### From git

```bash
sao sinchang/template-fastify-plugin my-project
```
